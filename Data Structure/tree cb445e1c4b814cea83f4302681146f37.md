# tree

# 용어

- **Root:** 트리의 시작점
- **Parent:** 한 노드를 가리키는 바로 위의 노드
- **Child:** 부모 노드에 의해 가리켜진 노드
- **Siblings:** 같은 부모를 공유하는 자식 노드들
- **Descendents:** 특정 노드의 자식, 자식의 자식 등을 포함한 모든 후손 노드들
- **Ancestors:** 특정 노드로부터 루트까지의 모든 상위 노드들, siblings는 제외된다.
- **Degree of a node:** 해당 노드의 자식 노드 수
- **Internal nodes / External nodes:** 자식이 있는 노드는 내부 노드, 자식이 없는 노드는 외부 노드
- **Levels:** 루트로부터 특정 노드까지의 경로 길이
- **Height:** 트리의 루트에서 가장 먼 잎까지의 경로 길이
- **Forest:** 트리의 집합, 여러 개의 트리로 이루어진 집합

# Binary Tree

: degree가 2이하인 tree이다. 즉, 모든 노드의 자식노드의 개수가 {0, 1, 2}라면, Binary Tree이다.

- **노드가 n개 주어졌을 때, 서로 다른  Binary Tree를 몇개 생성할 수 있을까?**
    
    **카탈란 수(Catalan number)**:2nCn / (n+1)
    

- **주어진 노드에 이름이 부여됬다면, n개 주어졌을때, 서로 다른 Binary Tree를 몇개 생성할 수 있을까?**
    
    :카탈란 수 * n!
    

- **노드가  n개 주어졌을 때, maximum height tree를 몇개 생성할 수 있을까?**
    
    max height tree의 개수: 2ⁿ⁻¹
    

- **높이 h가 주어졌을 때, 최소 노드개수와 최대 노드 개수는 무엇일까?**
    
    최소 노드개수:  n = h+1
    최대 노드개수:  n =$2^{n+1}$ - 1     //등비수열의 합꼴이다.
    
    **그럼 노드 n개가 주어졌을 때, 최소 높이와 최대 높이는 무엇일까?**
    최소 높이: h = $\log_{2}(n+1)$-1     //위의 최대 노드 개수 공식을 뒤집었다.
    최대높이: h = n-1     //위의 최소 노드개수를 뒤집었다.
    
    즉, binary tree의 노드 n은 h+1 ≤ n ≤ $2^{n+1}$ - 1 수식을 만족하고, 
    높이 h는  $\log_{2}(n+1)$-1 ≤ h ≤ n-1 수식을 만족한다고 할 수 있다.
    

- **Binary Tree에서 degree(2)와 degree(0)는 어떤 관계가 있을까?**
    
    : deg(0) = deg(2)+1     //deg(0)는 external node라고 볼 수 있다.
    

# Strict Binary Tree

:degree가 0과 2로만, 이루어진 Binary Tree이다.

- **높이 h가 주어졌을 때, 최소 노드개수와 최대 노드 개수는 무엇일까?**
    
    최소 노드 개수 n = 2h + 1
    최대 노드 개수 n = $2^{h+1}$ - 1     //등비수열의 합 꼴이다.
    
    노드 n이 주어졌을 때, 최소 높이와 최대 높이는 무엇일까?
    최소 높이 h = $\log_{2}(n+1)$-1
    최대 높이 h = (n-1)/2
    
    2h+1 ≤ n ≤ $2^{h+1}$ - 1
    
    $\log_{2}(n+1)$-1 ≤ h ≤ (n-1)/2
    

- **internal node(i)와 external node(e)의 개수사이에 어떤 관계가 있을까?**
    
    e = i + 1
    

# m-ary Tree

: degree가 m개 이하인 tree이다. m은 자식노드의 최대 개수를 뜻하고, ary는 접미사로, ‘차수’라는 뜻이다.

strict m-ary Tree라는 것도 있는데, 이는 degree가 0과 m으로만 이루어진 tree라는 뜻이다.

- **Strict m-ary Tree의 높이 h가 주어졌을 때, 최소 노드 개수와 최대 노드 개수는 무엇일까?**
    
    최소 노드 개수 n = m*h+1
    최대 노드 개수 n = ($m^{h+1}$-1) / (m-1)
    
    Strict m-ary Tree의 노드개수 n이 주어졌을 때, 최소 높이와 최대 높이는 무엇일까?
    최소 높이 h = $\log_{m}(n*(m-1)+1)-1$
    최대 높이 h = (n - 1) / m
    

- **Strict m-ary Tree의 internal node(e)와 external node(i)의 관계식은 무엇인가?**
    
    : e = (m-1)i+1
