# 표기법

고생한 파트.

# 표기법이란 무엇인가?

:표기법은 a+b와 같은 식을 어떻게 표기 할 것인가에 대한 방법이다. 표기법에는 3가지 방법이 있다.

- infix     : 연산자를 중심으로 양쪽에 피연산자가 위치한다. (우리가 쓰는 방식) [a+b]
 - prefix   : 연산자가 제일 앞에 오고 피연산자가 연달아 위치. [+ab]
 - postfix : 피연사자가 연달아 위치하고 연산자가 제일 뒤 [ab+]

# 왜 표기법을 사용하는가?

예를 들어 다음과 같은 수식이 있다고 생각해보자.

중위표기법(infix): 8+3*(9-6)/2^2+6/2

 수학에서 연산자들의 우선순위를 표로 정리하면 다음과 같다.

| 상위 | 연산자 | 설명 | 연산순서 |
| --- | --- | --- | --- |
| 1 | +, - | 덧셈, 뺄셈 연산 | L→R |
| 2 | *, /, % | 곱셈, 나눗셈, 나머지 연산 | L→R |
| 3 | 지수 ^ | 거듭제곱 연산 | R→L |
| 4 | 괄호 () | 괄호 안의 연산 우선 | L→R |

위 표에서 우선순위가 높은 연산자일수록 상위값이 커지도록 나타내었다. 괄호는 다른 모든 연산자보다 우선 순위가 높으며, 지수 연산자가 곱셈, 나눗셈, 나머지 연산보다 우선 순위가 높다.

연산자들에는 우선순위가 있기때문에, 컴퓨터의 입장에서는 infix방식으로 이해려면, 식을 여러번 스캔하면서 우선순위에 따라 앞뒤로 계속 움직이면서 계산을 해야한다. 이것은 매우 비효율적이다. 

따라서 수식을 후위표기법(postfix)으로 변형한다면, 한번의 스캔으로 식을 계산도록 할 수 있다.

후위표기법(postfix):  8396-22^/*+62/+

postfix방식의 계산은 앞에서부터 순차적으로 스캔을 하고, 연산자가 나오면, 바로 앞에있는 피연산자2개로 계산을 해준다. 그리고 계산된 결과값을 그대로 적고, 이 과정을 수식의 끝까지 반복해주면된다. 따라서 컴퓨터는 코드에 수식이 있으면, 논리적으로 infix를 postfix의 형태로 변환해 준다.

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

struct Node {
    int data;
    struct Node *next;
} *top = NULL;

void push(int x) {
    struct Node *t = (struct Node *)malloc(sizeof(struct Node));
    if (t == NULL)
        printf("stack is full\n");
    else {
        t->data = x;
        t->next = top;
        top = t;
    }
}

int pop() {
    int x = -1;
    struct Node *t = NULL;
    if (top == NULL)
        printf("stack is empty\n");
    else {
        t = top;
        x = t->data;
        top = top->next;
        free(t);
    }
    return x;
}

int IsOperand(char x) {
    if (x == '+' || x == '-' || x == '*' || x == '/')
        return 0;
    else
        return 1;
}

int pre(char x) {
    if (x == '+' || x == '-')
        return 1;
    else if (x == '*' || x == '/')
        return 2;
    return 0;
}

char *InToPostfix(char *Infix) {
    int i = 0, j = 0;
    int len = strlen(Infix);
    char *postfix = (char *)malloc(sizeof(char) * (len + 1));
    
    while (Infix[i] != '\0') {
        if (IsOperand(Infix[i]))
            postfix[j++] = Infix[i++];
        else {
            while (top != NULL && pre(top->data) >= pre(Infix[i]))
                postfix[j++] = pop();
            push(Infix[i++]);
        }
    }
    while (top != NULL) {
        postfix[j++] = pop();
    }
    postfix[j] = '\0';
    
    return postfix;
}

int main() {
    char *infix = "a+b-c+d*e/f-g";
    char *postfix = InToPostfix(infix);
    printf("%s\n", postfix);
    free(postfix);
    return 0;
}
```
